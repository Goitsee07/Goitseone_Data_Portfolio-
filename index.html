3. Tableau: COVID-19 Impact on South Africa's Health System

Key Questions:

  • How did hospital admissions change over time?
  • Which provinces were most overwhelmed?
  • What is the correlation between testing and hospitalizations?
  • Did mortality rates differ by age or gender?

Steps Taken:

  • Combined multiple CSV sources using Tableau Prep
  • Created interactive dashboards using filters and parameters
  • Designed maps and line charts for geo-temporal insights
  • Used calculated fields to normalize hospitalization data

Key Takeaways:

  • Wave peaks led to sharp surges in hospital admissions
  • Gauteng and WC saw the highest ICU strain
  • Mortality was highest in older age groups
